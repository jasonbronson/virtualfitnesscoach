Dynamic Drive jsDomenuBar demo I (absolutely positioned)

This is the demo of jsDomenuBar with the following settings:

  • Position is absolute (x: 10px, y: 10px) on the page
  • Theme is Windows XP

 The current folder ("dynamicdrive") is self containing, and contains all the files that make up this demo. Check out the configuration file ("ddconfig.inc.js") to get a feel on how to customize jsDomenuBar, and view the source of this page to see the necessary codes to insert on your page.

Obviously, be sure to also check out the other folders within this zip file for comprehensive documentation and demos as provided by the author himself.

Enjoy!
