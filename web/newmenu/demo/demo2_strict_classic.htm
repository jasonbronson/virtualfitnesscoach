The menu bar below has the Classic theme applied to it. You can change the theme of the menu bar and menus easily without reloading the page (works in certain browsers only).










To change to Office XP theme (without reloading the page), follow the following steps:

Mozilla 1.x/Netscape 7.x users:

  • From the menu toolbar, click View.
  • Go to Use Style.
  • Select Office XP.

Mozilla Firefox users:

  • From the menu toolbar, click View.
  • Go to Page Style.
  • Select Office XP.

Opera users:

  • From the menu toolbar, click View.
  • Go to Style.
  • Select Office XP.

Microsoft Internet Explorer/Others users:

  • Sorry, it does not work.
  • However, you can still view this page in Office XP theme by clicking here. (Page will reload.)
