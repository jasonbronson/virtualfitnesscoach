Sample: dhtmlxCombo Adding images to dhtmlxCombo dhtmlxCombo main page

Image Select

 
    <div id="combo_zone2" style="width:200px; height:30px;"></div>
    <script>
 
        var z=new dhtmlXCombo("combo_zone2","alfa2",200, 'image');
        z.setDefaultImage("../images/book.gif");
        z.loadXML("xml/data.xml");
</script>
    
    XML:
    <?xml version="1.0" ?>
    <complete>
        <option value="1">one</option>
        <option value="2" img_src="../images/books.gif">two</option>
        <option value="3" img_src="../images/books_cat.gif">three</option>
        <option value="4">four</option>
        ...
    </complete>
